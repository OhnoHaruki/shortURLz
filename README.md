# shortyURLz
URLを簡単に短く、簡潔にできます。

# 概要
URLを入力することでURLを簡単に短くすることができるサービスです。  
このサービスを使用することでURLの見栄えが良くなったり、SNS上にURLを載せる際に文字数の節約ができます。  
bit.lyのAPIを使用して作成します。  

# 使い方(Usage)
## コマンド

    Usage shortURLz<URL>[OPTION]
    OPTION:
    -h,--help   ヘルプメッセージの表示
    -v,--version    バージョンの表示
# インストール
    brew install shortURLz
